# WackyShell

WackyShell is a custom, interactive command-line shell in C# for experimenting with commands, file operations, and scripting ideas

## Features

Built-in commands: echo, cd, pwd, ls, mkdir, mkfile, rm, clear, exit, type

## Command explanation

echo: Prints out whatever the user typed after "echo" <br />
cd: Goes to specified directory <br />
pwd: Prints out current directory <br />
ls: lists all directories and files inside of current directory <br />
mkdir: creates a directory with the specified name <br />
mkfile: creates a file in the current directory with the specified name <br />
rm: Removes the specified directory or file <br />
clear: clears the command-line <br />
exit: quits the application <br /> 
type: used to check if a command is built-in. Example: type echo <br />
