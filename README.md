# WackyShell

WackyShell is a custom, interactive command-line shell in C# for experimenting with commands, file operations, and scripting ideas

## Features

Built-in commands: echo, cd, pwd, ls, mkdir, mkfile, rm, clear, exit, type

## Command explanation

echo: Prints out whatever the user typed after "echo"
cd: Goes to specified directory
pwd: Prints out current directory
ls: lists all directories and files inside of current directory
mkdir: creates a directory with the specified name
mkfile: creates a file in the current directory with the specified name
rm: Removes the specified directory or file
clear: clears the command-line
exit: quits the application
type: used to check if a command is built-in. Example: type echo
